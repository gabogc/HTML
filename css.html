Selectors

  • In line style="color:blue;"
  • Css Rule in Head withStyle tags , Selector and {}
  • .class only select elements that have been identified with a class
  • #element only selects elements marked with ID
  • Element plus Class selects only the elements with that id.
  • you can group selectors with comas.
